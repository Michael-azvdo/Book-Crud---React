# 📚 CRUD de Livros

Aplicação web desenvolvida em React que permite cadastrar, editar e excluir livros, simulando um sistema completo de gerenciamento.

---

## 🚀 Funcionalidades

- Adicionar novos livros
- Editar título e autor
- Excluir livros
- Persistência de dados com LocalStorage
- Atualização automática da interface
- Contador de livros cadastrados

---

## 🛠️ Tecnologias utilizadas

- React
- JavaScript
- HTML
- CSS
- LocalStorage

## ▶️ Como executar o projeto

1. Clone o repositório:

git clone https://github.com/Michael-azvdo/Book-Crud---React

2. Acesse a pasta:

cd react-books-crud

3. Instale as dependências:

npm install

4. Execute o projeto:

npm start
